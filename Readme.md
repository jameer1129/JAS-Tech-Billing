# JAS Tech Billing

A modern, lightweight, fully offline billing application built using **HTML, CSS, and JavaScript**.

JAS Tech Billing allows businesses to create professional invoices in seconds without requiring a backend or internet connection.

---

## Features

### Customer Management
- Customer Name
- Address
- Mobile Number

### Invoice Management
- Auto-generated Invoice Number
- Automatic Date Generation
- Professional Invoice Layout
- A4 PDF Export

### Product Management
- Add Products
- Edit Products
- Delete Products
- Quantity
- Price
- Description
- Serial Number
- Automatic Amount Calculation

### Invoice Summary
- Total Items
- Total Quantity
- Grand Total

### Notes
- Optional Notes Section
- Toggle Notes Visibility in PDF

### PDF Generation
- High Quality PDF
- A4 Size
- Company Logo
- Watermark
- QR Code
- Signature
- Footer
- Services Banner
- Auto Layout Adjustment

### UI Features
- Responsive Design
- Modern Interface
- Toast Notifications
- Confirmation Dialogs
- Product Modal
- Offline Support

## Configuration

Most company information can be changed from:

```
config.json
```

Example:

- Company Name
- Address
- Phone
- Email
- Theme Colors
- Logo
- Signature
- QR Code
- Footer
- Default Notes

No source code modification is required.
